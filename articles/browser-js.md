# history對象

history保存目前分頁的瀏覽記錄，是一array-like對象，其length屬性表示訪問過的網址。

`history.back()`      等同上一頁

`history.forward()`   等同下一頁

`history.go(1)`       等同下一頁

`history.go(0)`       等同重新整理
