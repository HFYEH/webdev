`ctrl+alt+a`	align codes
`ctrl+alt+j`	rearrange json
`crtl+shift+up/down`	move line
`ctrl+j`	join line
`ctrl+k+ctrl+u`	uppper case
`ctrl+k+ctrl+l`	lower case
`ctrl+r`	find function
`ctrl+i`	instant search
`ctrl+p...@`	find tags in the given file
