# Metaprogramming Ruby 代碼塊


代碼塊（Block）是控制作用域（Scope）的手段。

Block僅是可調用對象的一員，其他還有像proc，lambda這樣的對象。

目前為止講的都是Ruby物件導向的特性，然而在這一章講的block，觀念是繼承於functional programming language。
